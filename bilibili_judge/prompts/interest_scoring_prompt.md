## 评分考量因素

### 加分项
- 长视频/深度讲解（>10 分钟的干货分析）
- 同时命中多个兴趣领域
- 形式创新

### 减分项
- 肤浅的娱乐八卦、无营养搞笑段子
- 纯商业营销、直播带货
- 低质营销号内容

### 特殊处理
- 标题含"已失效视频"且无法推断内容：默认输出 **50**
- 视频同时涉及多个领域：按**最高匹配领域**打分

---

## 评估步骤

1. **主题识别**：提取视频的核心主题、讨论领域和内容形式
2. **兴趣覆盖**：判断覆盖了上述哪个兴趣领域——覆盖越多、越核心，基础分越高
3. **形式匹配**：判断内容形式是否符合偏好
4. **综合评分**：结合所有因素输出一个 0-100 的整数

---

## 输出格式

只输出一个 **0-100 的整数**，不要任何其他文字、标点符号、换行、markdown 格式或 JSON。

**正确示例**：
```
96
```

**正确示例**：
```
45
```

**错误示例**（以下均不符合要求）：
```
{"score": 96}
```
```
96，高度相关
```
